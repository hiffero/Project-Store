# 🛍️ Toko Bobby App

A modern **E-Commerce Mobile Application** built using Flutter, powered by a custom RESTful API with Node.js and MySQL.

---

## 📱 Overview

Toko Bobby App is a mobile application designed to simplify online shopping experiences.
Users can browse products, add items to cart, and perform checkout seamlessly.

---

## ✨ Features

* 🛒 Product listing
* 🔍 Product detail view
* ➕ Add to cart
* 💳 Checkout system
* 🔐 Authentication (Login/Register)
* 🌐 API integration (RESTful)

---

## 🛠️ Tech Stack

### Frontend

* Flutter (Dart)
* Material UI

### Backend

* Node.js (Express.js)
* MySQL Database

## ⚙️ Installation

### 1. Clone Repository

```
git clone https://github.com/username/toko_bobby.git
cd toko_bobby
```

---

### 2. Backend Setup

```
cd backend
npm install
```

Create `.env` file:

```
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=toko_bobby
PORT=3000
```

Run server:

```
npm start
```

---

### 3. Flutter Setup

```
cd flutter_app
flutter pub get
flutter run
```
